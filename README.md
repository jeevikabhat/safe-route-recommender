# Safe Route Recommender рЯЪґвАНвЩАпЄП

A web application that recommends safer travel routes based on crowd-sourced data.

## Features
- Calculate safety score using SQL
- Submit safety reports
- Compare multiple routes

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript

## How it works
Safety score is calculated using:
- Lighting conditions
- Crowd density
- Incident reports
